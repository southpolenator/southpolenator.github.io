StackFrame Properties

The StackFrame type exposes the following members.

Properties
  NameDescription
Arguments
Gets the function arguments.
Current
Gets or sets the current stack frame in current thread of current process.
FrameContext
Gets the frame context.
FrameNumber
Gets the index of the frame. This index counts the number of frames from the top of the call stack. The frame at the top of the stack, representing the current call, has index zero..
FrameOffset
Gets the location in the process's virtual address space of the stack frame, if known. Some processor architectures do not have a frame or have more than one. In these cases, the engine chooses a value most representative for the given level of the stack.
FunctionDisplacement
Gets the function displacement.
FunctionName
Gets the name of the function (including module name).
FunctionNameWithoutModule
Gets the function name without module name.
InstructionOffset
Gets the location in the process's virtual address space of the related instruction for the stack frame. This is typically the return address for the next stack frame, or the current instruction pointer if the frame is at the top of the stack.
Locals
Gets the local variables.
Process
Gets the owning process.
ReturnOffset
Gets the location in the process's virtual address space of the return address for the stack frame. This is typically the related instruction for the previous stack frame.
SourceFileDisplacement
Gets the source file displacement.
SourceFileLine
Gets the source file line.
SourceFileName
Gets the name of the source file.
StackOffset
Gets the location in the process's virtual address space of the processor stack.
StackTrace
Gets the owning stack trace.
Thread
Gets the owning thread.
Virtual
Gets a value indicating whether this StackFrame is virtual.
