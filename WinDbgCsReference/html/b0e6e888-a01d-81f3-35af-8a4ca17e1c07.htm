GenericsElementCasterT Class

Helper class to be used in Generics User Type classes as partial class extension. It is needed when Generics User Type casts variable to new user type (that is template and possible generics). When user type is generics, engine cannot deduce CodeType that needs to be used, while generated user type code expects correct one. This class provides the bridge between two worlds.

Examples
When you have exported user type that is generics
Copy
MyGenericsType<T>
you should use this class to solve problems that can arise of having this somewhere in the code:
Copy
MyGenericsType<MySecondGenericsType<int>>
. Using generics element caster solves problems by extracting inner user type from the code type and later using it when casting.
