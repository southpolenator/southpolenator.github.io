Version 1.0.0.3

Version 1.0.0.3 was released on 2/7/2016.

Changes in This Release

  • Adding new option to GenerateUserTypesFromPdb to always use new instead of user type casting.

  • Adding new option to GenerateUserTypesFromPdb to compile generated code into assembly.

  • Adding support in DIA symbol provider for reading locals that have LocationType static.

  • UserMemberTToString now forwards call to cached value's ToString() function.

  • GenerateUserTypesFromPdb generates properties for getting base classes for Multiple inheritance.

  • Adding support for enums and GenerateUserTypesFromPdb now exports all enums used in exported UserTypes.

  • Adding support for support for double/float variables and adding casting operators for them.

  • Adding support for accessing base classes from CodeType

  • Easy conversion from CodePointerT to CodeArrayT.

  • Debugger now has function for searching memory for pattern:

    • FindBytePatternInMemory

    • FindAllBytePatternInMemory

    • FindPatternInMemory

    • FindAllPatternInMemory

    • FindTextPatternInMemory

    • FindAllTextPatternInMemory

  • Bug fixes

    • Reading static array members

    • Code generation will now always generates constructor needed for base class constructor

    • VariableCastAsT works with UserType types that haven't been loaded into metadata cache.

    • Code generation now calls .ToString() instead of type casting to string.

    • VariableVariable(Variable) now copies all member fields.

    • VariableGetName now has full path to how user get to this variable.
