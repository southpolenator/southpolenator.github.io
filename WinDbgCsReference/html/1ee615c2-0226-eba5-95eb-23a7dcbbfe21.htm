ProcessSwitcher
Used for scoped process switching.
Examples
Copy
using (var switcher = new ProcessSwitcher(process))
{
    // Invoke DbgEng.dll interface function
}
Remarks
Use this class for accessing process information from DbgEng.dll interfaces to insure correct process information access.
Remarks
For performance reasons, after using scope, previous process won't be set until it is needed. Always use this class to insure correctness.
