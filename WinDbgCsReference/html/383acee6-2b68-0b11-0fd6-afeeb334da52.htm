DefaultObjectWriter
Helper class for writing objects during interactive scripting. Set InteractiveScriptBase.ObjectWriter to change the effects of interactive scripting.
Executor
Exported class for transferring execution from native to managed code
GenericsElementCasterT
Helper class to be used in Generics User Type classes as partial class extension. It is needed when Generics User Type casts variable to new user type (that is template and possible generics). When user type is generics, engine cannot deduce CodeType that needs to be used, while generated user type code expects correct one. This class provides the bridge between two worlds.
InteractiveScriptBase
Base class for interactive script commands
InteractiveScriptBaseExtensions
Helper for dumping objects using InteractiveScriptBase.ObjectWriter.

Module of the debugging process.
NakedPointer

UserType
Base class for user defined types in C# scripts
UserTypeAttribute
Attribute that gives more info for user defined types in scripts.
Variable
Ultimate class for working with variables from process being debugged.
VariableCastExtender
Extension class providing cast functionality.
VariableCollection
This is helper class for easier navigation through local variables and arguments. It is not performance optimized and should not be used as dictionary, but only as array of variables.
