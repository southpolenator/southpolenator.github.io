GetTypeByName Method

IClrModuleGetTypeByName Method

Attempts to obtain a IClrType based on the name of the type. Note this is a "best effort" due to the way that the DAC handles types. This function will fail for Generics, and types which have never been constructed in the target process. Please be sure to null-check the return value of this function.
