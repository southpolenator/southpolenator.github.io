CsDebugScript.Engine.Utility Namespace

Collection of utility classes.
Classes
  ClassDescription
CacheInvalidator
Helper method which invalidates all class fields which are caches starting from the given root objects.
DebuggerOutputToTextWriter
Helper class for capturing debugger output while executing commands. Example usage:

using (var callbacks = DebuggerOutputToTextWriter.Create(textWriter, captureFlags) { }

DictionaryCacheTKey, TValue
Helper class for caching objects inside the dictionary. New object will be cached on the request.
DumpFileMemoryReader
Wraps functionality for reading files that are region mapped. [Address start, Address end] maps to a region in the file. Dumps are usually like that, hence the class name.
OutputCallbacksSwitcher
Helper class for replacing debugger output callbacks - used when capturing command output. Example usage:

using (var switcher = OutputCallbacksSwitcher.Create(outputCallbacks) { }

SimpleCache
Helper class for caching results - it is being used as lazy evaluation

Helper class for caching results - it is being used as lazy evaluation
