StackFrameSwitcher Class
Used for scoped stack frame switching.

    // Invoke DbgEng.dll interface function
}
Remarks
Use this class for accessing stack frame information from DbgEng.dll interfaces to insure correct stack frame information access.
Remarks
For performance reasons, after using scope, previous stack frame won't be set until it is needed. Always use this class to insure correctness.
